Swift рЃерЃў рЃўрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА callback рЃў рЃњрЃљрЃЋрЃљрЃДрЃЮрЃџрЃЮрЃЌ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ Closure рЃА. Closure рЃљрЃарЃўрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃЉрЃџрЃЮрЃЎрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ. Closure рЃћрЃЉрЃўрЃљ рЃўрЃњрЃўрЃЋрЃћрЃљ рЃарЃљрЃф рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ block рЃћрЃЉрЃў C рЃерЃў, Obj-C рЃерЃў рЃЊрЃљ lambda рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў рЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљ рЃћрЃюрЃћрЃЉрЃерЃў.

Closure рЃћрЃЉрЃўрЃА рЃФрЃљрЃџрЃљ рЃўрЃЏрЃљрЃџрЃћрЃЉрЃљ Capturing рЃерЃў. Capturing рЃў рЃљрЃарЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃарЃЮрЃфрЃљ рЃќрЃћрЃЏрЃЮрЃЌ рЃ«рЃАрЃћрЃюрЃћрЃЉрЃБрЃџрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃА рЃДрЃЋрЃћрЃџрЃљ reference, рЃЎрЃЮрЃюрЃАрЃбрЃљрЃюрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃўрЃЏ рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃерЃў рЃАрЃљрЃЊрЃљрЃф Closure рЃў рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ.

рЃЋрЃюрЃљрЃ«рЃЮрЃЌ рЃърЃљрЃбрЃљрЃарЃљ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃерЃў рЃЋрЃ«рЃћрЃЊрЃљрЃЋрЃЌ f closure рЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф Capturing рЃА рЃБрЃЎрЃћрЃЌрЃћрЃЉрЃА Environment рЃА рЃАрЃљрЃЊрЃљрЃф firstVariable рЃфрЃЋрЃџрЃљрЃЊрЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА.

*   Capturing - рЃљрЃарЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА context рЃерЃў рЃЏрЃДрЃЮрЃц рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃА.
*   Environment - Closure рЃўрЃА рЃњрЃљрЃарЃћрЃЌ рЃЏрЃДрЃЮрЃцрЃў context рЃў, рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃў, рЃарЃћрЃцрЃћрЃарЃћрЃюрЃАрЃў рЃЊрЃљ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃарЃЮрЃЏрЃџрЃўрЃА Capturing рЃўрЃф рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ.

closure рЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃљрЃерЃўрЃю рЃ«рЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф f рЃА рЃЊрЃљрЃЋрЃБрЃФрЃљрЃ«рЃћрЃЉрЃЌ, рЃарЃљрЃф рЃўрЃЏрЃљрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃарЃЮрЃЏ Captured рЃфрЃЋрЃџрЃљрЃЊрЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃЊрЃарЃЮрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃЮрЃА, рЃњрЃљрЃЏрЃЮрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃљрЃЦрЃћрЃЊрЃљрЃю Closure рЃўрЃА рЃърЃљрЃАрЃБрЃ«рЃўрЃф рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ.

Closure рЃћрЃЉрЃА reference рЃАрЃћрЃЏрЃљрЃюрЃбрЃўрЃЎрЃљ рЃљрЃЦрЃЋрЃА рЃЊрЃљ рЃљрЃарЃљ value. рЃљрЃюрЃБ Closure рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў heap рЃерЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃЮрЃцрЃљ рЃЊрЃљ рЃљрЃарЃљ stack рЃерЃў, рЃарЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў ? рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃарЃљрЃф Environment рЃерЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ, рЃгрЃљрЃЋрЃљ heap рЃерЃў Closure рЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ.

рЃќрЃћрЃЏрЃЮрЃЌ рЃ«рЃАрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ, рЃ«рЃерЃўрЃарЃўрЃљ рЃўрЃЏрЃўрЃА рЃарЃўрЃАрЃЎрЃў рЃарЃЮрЃЏ Closure рЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА рЃњрЃљрЃњрЃЋрЃћрЃърЃљрЃарЃЮрЃА Memory leak рЃў. рЃљрЃЏрЃўрЃА рЃърЃарЃћрЃЋрЃћрЃюрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃарЃљрЃЏрЃЮрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃ«рЃћрЃарЃ«рЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА, рЃЌрЃБрЃЏрЃфрЃљ рЃЏрЃљрЃюрЃљрЃЏрЃЊрЃћ рЃЋрЃўрЃАрЃљрЃБрЃЉрЃарЃЮрЃЌ Closure рЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃќрЃћ Swift рЃерЃў.

рЃАрЃБрЃџ рЃЮрЃарЃў рЃбрЃўрЃърЃўрЃА Closure рЃў рЃњрЃЋрЃљрЃЦрЃЋрЃА.

@nonescaping Closure рЃћрЃЉрЃў

рЃћрЃАрЃћрЃЌрЃўрЃљ Closure рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃ«рЃЋрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃљрЃЊ рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљ рЃљрЃЏрЃљрЃЋрЃћ function body рЃерЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃарЃћрЃќрЃБрЃџрЃбрЃљрЃбрЃўрЃф рЃЏрЃљрЃерЃўрЃюрЃЋрЃћ рЃЉрЃарЃБрЃюрЃЊрЃћрЃЉрЃљ. рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃњрЃљрЃЊрЃљрЃгрЃЮрЃЊрЃћрЃЉрЃБрЃџрЃў Closure рЃў out of scope рЃљрЃдрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ, рЃЏрЃўрЃАрЃў reference counter рЃў 0 рЃњрЃљрЃ«рЃЊрЃћрЃЉрЃљ рЃЊрЃљ [ARC](https://docs.swift.org/swift-book/LanguageGuide/AutomaticReferenceCounting.html) рЃгрЃљрЃерЃџрЃўрЃА рЃЏрЃљрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃЊрЃљрЃю.

nonescaping Closure рЃўрЃА рЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃўрЃА рЃБрЃюрЃљрЃарЃўрЃљрЃюрЃЮрЃЉрЃљ рЃћрЃАрЃћрЃЌрЃўрЃљ:
1. рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃљрЃЊ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА (heap рЃерЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃЮрЃцрЃљ рЃљрЃЊрЃњрЃўрЃџрЃў)
2. Environment capturing рЃўрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃарЃљрЃўрЃЏрЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃАрЃљрЃф рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА.
3. рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃћрЃФрЃљрЃ«рЃўрЃА Closure рЃА.
4. рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃА, Closure рЃњрЃљрЃАрЃфрЃЊрЃћрЃЉрЃљ scope рЃА рЃЊрЃљ рЃгрЃљрЃўрЃерЃџрЃћрЃЉрЃљ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃЊрЃљрЃю.

рЃљрЃЏ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃерЃў completionHandler рЃў nonescaping Closure рЃўрЃљ, рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃўрЃА рЃЏрЃўрЃгрЃћрЃарЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃљрЃа рЃљрЃарЃўрЃА, рЃарЃљрЃЊрЃњрЃљрЃю рЃДрЃЋрЃћрЃџрЃљ Closure рЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф Environment рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃљрЃАрЃарЃБрЃџрЃљрЃЏрЃЊрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћрЃЉрЃљ default рЃљрЃЊ рЃљрЃарЃўрЃА nonescaping рЃў.

**рЃарЃЮрЃЏрЃћрЃџрЃўрЃљ Environment рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ ?**

*   getSumOfArray

**рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћрЃЉрЃљ completioHandler рЃў рЃўрЃЦрЃљрЃЏрЃЊрЃћ рЃАрЃљрЃюрЃљрЃЏ getSumOfArray рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ ?**

*   рЃЎрЃў

рЃњрЃљрЃЏрЃЮрЃЊрЃўрЃА, рЃарЃЮрЃЏ рЃАрЃљрЃюрЃћрЃарЃЋрЃўрЃБрЃџрЃЮ рЃљрЃарЃљрЃцрЃћрЃарЃў рЃЏрЃљрЃЦрЃЋрЃА, рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЋрЃљрЃа рЃарЃЮрЃЏ рЃЊрЃарЃЮрЃћрЃЉрЃўрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў reference рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃўрЃЦрЃљрЃЏрЃЊрЃћ рЃгрЃљрЃўрЃерЃџрЃћрЃЉрЃљ рЃАрЃљрЃюрЃљрЃЏ Environment рЃў рЃфрЃЮрЃфрЃ«рЃљрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ.

@escaping Closure рЃћрЃЉрЃў
=====================

рЃарЃЮрЃЊрЃћрЃАрЃљрЃф Closure рЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃљрЃЊ рЃњрЃљрЃЊрЃљрЃЋрЃфрЃћрЃЏрЃЌ рЃЊрЃљ рЃЋрЃўрЃфрЃўрЃЌ, рЃарЃЮрЃЏ function body рЃБрЃцрЃарЃЮ рЃЏрЃљрЃџрЃћ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃљрЃа рЃЋрЃўрЃфрЃўрЃЌ Closure рЃў рЃќрЃБрЃАрЃбрЃљрЃЊ рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћрЃЉрЃљ рЃћрЃАрЃћрЃЌ рЃЊрЃарЃЮрЃА рЃњрЃЋрЃўрЃгрЃћрЃЋрЃА рЃарЃЮрЃЏ Closure рЃЏрЃЮрЃЋрЃюрЃўрЃерЃюрЃЮрЃЌ @escaping рЃљрЃЊ.

рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА execution рЃў рЃЏрЃЮрЃарЃЕрЃћрЃЉрЃљ, Closure рЃЏрЃљрЃўрЃюрЃф рЃўрЃљрЃарЃАрЃћрЃЉрЃћрЃЉрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў рЃўрЃЦрЃљрЃЏрЃЊрЃћ рЃАрЃљрЃюрЃљрЃЏ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљ рЃљрЃа рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃљ.

**рЃарЃЮрЃЊрЃўрЃА рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ escaping Closure рЃў?**

*   рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃў Execution рЃўрЃА рЃЊрЃарЃЮрЃА.

1.  рЃарЃЮрЃЊрЃћрЃАрЃљрЃф Closure рЃА рЃЋрЃљрЃбрЃљрЃюрЃЌ DispatchQueue рЃА рЃљрЃю рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃўрЃАрЃћрЃЌ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃарЃљрЃўрЃЏрЃћ рЃАрЃљрЃ«рЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃА рЃБрЃерЃЋрЃћрЃЉрЃА background thread рЃќрЃћ рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃърЃўрЃарЃћрЃЉрЃА рЃЕрЃЋрЃћрЃюрЃў Closure рЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљрЃА, рЃарЃЮрЃЏ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃА рЃарЃћрЃќрЃБрЃџрЃбрЃљрЃбрЃў рЃЊрЃљрЃњрЃЋрЃўрЃЉрЃарЃБрЃюрЃЮрЃА.
2.  рЃљрЃю рЃЕрЃЋрЃћрЃюрЃА рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃБрЃџ Closure рЃА рЃБрЃерЃЋрЃћрЃЉрЃА background thread рЃќрЃћ рЃЊрЃљ рЃљрЃЦрЃљрЃф рЃљрЃа рЃЋрЃўрЃфрЃўрЃЌ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃарЃЮрЃЊрЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ.

рЃДрЃЋрЃћрЃџрЃљ рЃћрЃАрЃћрЃЌ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў Environment рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃљрЃърЃарЃўрЃЮрЃарЃерЃў рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃўрЃЦрЃљрЃЏрЃЊрЃћ рЃАрЃљрЃюрЃљрЃЏ рЃЕрЃЋрЃћрЃюрЃў closure рЃў рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћрЃЉрЃљ, рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃљрЃа рЃЋрЃўрЃфрЃўрЃЌ рЃћрЃА рЃарЃЮрЃЊрЃўрЃА рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃљ. рЃљрЃАрЃћрЃЌ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃерЃў Compiler рЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃћрЃЉрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА, рЃљрЃЦрЃћрЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЏрЃљрЃА рЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа, рЃарЃЮрЃЏ рЃўрЃфрЃЮрЃЊрЃћрЃА рЃарЃЮрЃЏрЃћрЃџрЃў Closure рЃўрЃАрЃљ escaping рЃЊрЃљ nonescapig рЃў.

рЃљрЃЏ рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃўрЃЌ рЃ«рЃЋрЃЊрЃћрЃЉрЃљ compiler рЃў рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџрЃў рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃўрЃЌ рЃљрЃљрЃЏрЃБрЃерЃљрЃЮрЃА рЃЕрЃЋрЃћрЃюрЃў рЃЎрЃЮрЃЊрЃў.

рЃЌрЃБ Closure рЃА рЃарЃљрЃўрЃЏрЃћ рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃЊрЃљ рЃЏрЃљрЃА @escaping рЃА рЃљрЃа рЃЏрЃўрЃЋрЃБрЃгрЃћрЃарЃЌ, Compile time error рЃў рЃњрЃЋрЃћрЃЦрЃюрЃћрЃЉрЃљ.

рЃарЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃўрЃЮрЃА Closure рЃћрЃЉрЃЏрЃљ ?
================================================

рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃЋрЃдрЃюрЃўрЃерЃюрЃћрЃЌ Closure рЃў reference type рЃўрЃљ, рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ heap рЃерЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃЮрЃцрЃљ. Capturing рЃўрЃА рЃЊрЃарЃЮрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃЌрЃБ рЃерЃћрЃю self рЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉ closure рЃерЃў, рЃўрЃА self рЃўрЃА strong reference рЃА рЃерЃћрЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃўрЃАрЃў рЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў. рЃЌрЃБ self рЃА рЃљрЃАрЃћрЃЋрЃћ рЃЊрЃљрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃарЃЮрЃЏ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃА closure рЃўрЃА рЃарЃћрЃцрЃћрЃарЃћрЃюрЃАрЃў (рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЌрЃБ рЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљрЃА рЃЏрЃЮрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃўрЃЌ рЃњрЃћрЃњрЃЏрЃљрЃЋрЃА) рЃЏрЃљрЃерЃўрЃю рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА retain cycle рЃљрЃюрЃБ circular reference рЃў рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃљ.

[retain cycle](https://vinodhswamy.medium.com/strong-cycle-retain-cycle-in-swift-f452f07518b2) рЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ memory leak рЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА.

рЃарЃЮрЃњрЃЮрЃа рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃЋрЃўрЃфрЃЋрЃљрЃЌ рЃЌрЃљрЃЋрЃў ? рЃЏрЃљрЃЊрЃџрЃЮрЃЉрЃљ [Chris lattner](https://en.wikipedia.org/wiki/Chris_Lattner) рЃА, рЃарЃЮрЃЏ рЃњрЃЋрЃљрЃЦрЃЋрЃА weak рЃЊрЃљ unowned keyword рЃћрЃЉрЃў.

рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃарЃЮрЃЏ рЃЋрЃЌрЃЦрЃЋрЃљрЃЌ weak рЃЊрЃљ unowned рЃў рЃћрЃБрЃЉрЃюрЃћрЃЉрЃљ ARC рЃўрЃА, рЃарЃЮрЃЏ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ reference counter рЃў рЃљрЃа рЃњрЃљрЃўрЃќрЃљрЃарЃЊрЃЮрЃА рЃЊрЃљ рЃўрЃА рЃЏрЃБрЃЊрЃљрЃЏ 1 рЃўрЃДрЃЮрЃА. рЃћрЃА рЃљрЃдрЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃћрЃЋрЃА strong circular reference cycle рЃА closure рЃАрЃљ рЃЊрЃљ self рЃА рЃерЃЮрЃарЃўрЃА. (рЃБрЃцрЃарЃЮ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃљрЃЊ ARC рЃќрЃћ рЃЊрЃљ memory management рЃќрЃћ, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃбрЃљрЃбрЃўрЃћрЃЉрЃерЃў рЃЋрЃўрЃАрЃљрЃБрЃЉрЃарЃћрЃЉ)

рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ weak рЃАрЃљ рЃЊрЃљ unowned рЃерЃЮрЃарЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃфрЃўрЃарЃћрЃљ

=========================================

рЃЏрЃћрЃЏрЃњрЃЮрЃюрЃў рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃАрЃќрЃћ рЃЏрЃћрЃбрЃў рЃЋрЃўрЃАрЃљрЃБрЃЉрЃарЃћ Closure рЃћрЃЉрЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃќрЃћ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃљрЃўрЃюрЃф рЃЏрЃњрЃЮрЃюрЃўрЃљ, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃюрЃљрЃЌрЃџрЃљрЃЊ рЃЏрЃљрЃерЃўрЃю рЃЋрЃўрЃњрЃћрЃЉрЃЌ рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљрЃА, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЕрЃЋрЃћрЃюрЃўрЃЌрЃЋрЃћ рЃЋрЃЦрЃЏрЃюрЃўрЃЌ рЃЏрЃљрЃЌ.

Closure рЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЮрЃа рЃљрЃќрЃарЃЮрЃЋрЃљрЃюрЃўрЃљ рЃЊрЃљ рЃЏрЃЮрЃЊрЃўрЃЌ рЃћрЃАрЃћ рЃњрЃљрЃЋрЃљрЃюрЃљрЃгрЃўрЃџрЃЮрЃЌ.

2.  рЃЏрЃћрЃЮрЃарЃћ рЃюрЃљрЃгрЃўрЃџрЃерЃў, рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃњрЃљрЃЋрЃўрЃЏрЃћрЃЮрЃарЃЮрЃЌ рЃўрЃА рЃарЃљрЃф Memory рЃерЃў рЃ«рЃЊрЃћрЃЉрЃљ, Capturing рЃўрЃА рЃЊрЃарЃЮрЃА рЃЊрЃљ рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ рЃњрЃљрЃЋрЃљрЃЎрЃћрЃЌрЃЮрЃЌ, рЃљрЃЏрЃљрЃА рЃЎрЃў Complex part

рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџ рЃЎрЃЮрЃЊрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃА рЃЌрЃБ рЃЎрЃљрЃарЃњрЃљрЃЊ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃљрЃЋрЃЌ, рЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃарЃЮрЃЏ рЃЕрЃЋрЃћрЃюрЃў рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃў рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ рЃЏрЃерЃЋрЃћрЃюрЃўрЃЋрЃарЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА. рЃќрЃБрЃАрЃбрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃЌ memory рЃерЃў рЃљрЃЊрЃњрЃўрЃџрЃА рЃЊрЃљ Capturing рЃў рЃАрЃљрЃЊрЃљрЃф рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў value рЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃњрЃЋрЃўрЃюрЃЊрЃљ рЃЏрЃерЃЋрЃћрЃюрЃўрЃЋрЃарЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА.

рЃАрЃўрЃЏрЃБрЃџрЃљрЃфрЃўрЃБрЃарЃў Closure рЃўрЃАрЃЌрЃЋрЃўрЃА рЃќрЃБрЃАрЃбрЃљрЃЊ рЃљрЃцрЃЊрЃћрЃўрЃЌрЃЊрЃћрЃЉрЃљ Box рЃерЃў рЃерЃћрЃюрЃљрЃ«рЃБрЃџрЃў рЃњрЃљрЃарЃћ рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў.

рЃЎрЃЮрЃЊрЃА рЃЌрЃБ рЃЎрЃљрЃарЃњрЃљрЃЊ рЃЊрЃљрЃљрЃЎрЃЋрЃўрЃарЃЊрЃўрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃ«рЃљрЃќрЃў рЃЊрЃљрЃЎрЃЮрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃњрЃљрЃарЃЊрЃљ рЃћрЃарЃЌрЃўрЃАрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃю рЃАрЃљрЃЦрЃЏрЃћрЃА, Capturing рЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА 1 memory рЃЊрЃљрЃю рЃЏрЃћрЃЮрЃарЃћрЃќрЃћ.

withMemoryRebound рЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃњрЃљрЃБрЃерЃЋрЃљрЃА рЃњрЃљрЃЊрЃљрЃгрЃЮрЃЊрЃћрЃЉрЃБрЃџрЃў closure рЃў, рЃАрЃљрЃЎрЃБрЃЌрЃљрЃа рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃерЃў рЃўрЃАрЃћ рЃарЃЮрЃЏ ThickFunction рЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃў рЃърЃЮрЃўрЃюрЃбрЃћрЃарЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃА binding рЃў рЃњрЃљрЃБрЃЎрЃћрЃЌрЃЮрЃА рЃЊрЃљ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃљрЃЊ рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃЮрЃА. рЃќрЃБрЃАрЃбрЃљрЃЊ рЃћрЃАрЃћ рЃ«рЃЊрЃћрЃЉрЃљ Capturing рЃў рЃцрЃљрЃарЃЊрЃўрЃА рЃБрЃЎрЃљрЃю. рЃћрЃарЃЌрЃў Environment рЃўрЃЊрЃљрЃю рЃЏрЃћрЃЮрЃарЃћ рЃЏрЃћрЃЏрЃЮрЃарЃўрЃерЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљрЃА рЃ»рЃћрЃа рЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЮрЃарЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА binding рЃў, рЃ«рЃЮрЃџрЃЮ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃљ рЃЊрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃћрЃарЃЌрЃў рЃљрЃЊрЃњрЃўрЃџрЃўрЃЊрЃљрЃю рЃЏрЃћрЃЮрЃарЃћрЃерЃў.

рЃЎрЃљрЃарЃњрЃў рЃћрЃ«рЃџрЃљ рЃњрЃљрЃЋрЃўрЃљрЃќрЃарЃЮрЃЌ binding рЃў рЃарЃљ рЃљрЃарЃўрЃА ?

C/C++ рЃерЃў memory рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃўрЃАрЃћрЃЌ рЃюрЃљрЃгрЃўрЃџрЃА рЃћрЃгрЃЮрЃЊрЃћрЃЉрЃљ рЃарЃЮрЃЏрЃћрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃарЃљрЃдрЃљрЃф рЃфрЃЋрЃџрЃљрЃЊрЃерЃў рЃгрЃћрЃарЃўрЃљ. C/C++рЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃЌрЃљрЃЋрЃљрЃЊ рЃбрЃўрЃърЃўрЃА рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљрЃАрЃљрЃф рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА, рЃўрЃЏрЃўрЃбрЃЮ рЃарЃЮ staticly typed рЃћрЃюрЃљрЃљ. рЃБрЃцрЃарЃЮ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ, рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃарЃљрЃдрЃљрЃф рЃЏрЃЮрЃюрЃљрЃЎрЃЋрЃћрЃЌрЃерЃў рЃгрЃћрЃарЃўрЃљ рЃЉрЃљрЃўрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃљрЃЏ рЃЉрЃљрЃўрЃбрЃћрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃбрЃўрЃърЃўрЃА рЃфрЃЮрЃЊрЃюрЃљ. рЃќрЃБрЃАрЃбрЃљрЃЊ рЃћрЃарЃЌрЃўрЃЊрЃљрЃўрЃњрЃўрЃЋрЃћ рЃЉрЃўрЃбрЃћрЃЉрЃўрЃА рЃЏрЃгрЃЎрЃарЃўрЃЋрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЮрЃа рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃарЃљрЃдрЃљрЃфрЃљрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃЊрЃћрЃА рЃўрЃЏрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃЌрЃБ рЃарЃљ рЃбрЃўрЃърЃерЃў рЃЕрЃљрЃгрЃћрЃа. рЃЌрЃљрЃю, рЃбрЃўрЃърЃўрЃА рЃфрЃЮрЃЊрЃюрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃА рЃЋрЃћрЃа рЃЕрЃљрЃљрЃбрЃљрЃарЃћрЃЉ рЃЏрЃљрЃњ рЃЏрЃЮрЃюрЃљрЃќрЃћрЃЏрЃќрЃћ.

рЃљрЃюрЃБ, C/C++рЃерЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА рЃљрЃЦрЃЋрЃА рЃЮрЃарЃў рЃарЃљрЃдрЃљрЃфрЃљ: рЃЉрЃљрЃўрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃбрЃўрЃърЃў. рЃЉрЃљрЃўрЃбрЃћрЃЉрЃў рЃарЃЮ value рЃўрЃфрЃЮрЃЊрЃћрЃА рЃЎрЃЮрЃЏрЃърЃўрЃџрЃћрЃарЃЏрЃљ рЃЊрЃљ рЃбрЃўрЃърЃў рЃарЃЮ рЃљрЃЏ value-рЃќрЃћ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃЕрЃљрЃљрЃбрЃљрЃарЃЮрЃА.
